# LoRaMesh

A discrete event simulator for LoRa based mesh network. Based on LoRaSim.

## Environment

* Python 3.5
* SymPy 3.0

## Realised

* DSDV
* asynchronous p-CSMA

## To Realise

* TDMA with centralilzed scheduling

## Future work

* Clustering by SF/frequency
* Adaptive link
* Visualization
